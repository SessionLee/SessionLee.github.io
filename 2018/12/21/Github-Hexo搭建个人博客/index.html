Github+Hexo搭建个人博客

发表于 2018-12-21 | 分类于 学习笔记

前言

突然很想找个地方记录和分享一下学习笔记和感悟，于是萌生了一个搭建个人博客的想法。

对比了CSDN和博客园等等现有的平台，觉得都不太适合。无意中找到了一个Github+Hexo的解决方案，觉得很适合，遂记录下来。

软件准备

我个人的Windows开发机上是没有装Nodejs和Git的，好在Windows平台有现成的安装程序，很简单。

Nodejs选择了当前（2018年12月21日）最新版本的LTS 10.14.2，直接在官网 下载就可以

Git下载地址：https://git-scm.com/download/win

考虑到要使用Github，于是另外安装了Github Desktop。下载地址：https://desktop.github.com/

配置Git

在https://github.com/settings/keys 中创建一个新的SSH Key，并将刚刚生成的id_rsa.pub文件内容复制进去。

如果使用Github Desktop自带的Git程序，则不需要进行上述两步。

创建个人仓库

在github上创建一个新的仓库，名字为用户名.github.io。如果用Github Desktop的话，需要额外点击Publish来发布到Github上。
